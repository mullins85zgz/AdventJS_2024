## Getting Started

To get started with this project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   cd js-challenges
   ```

2. Install the dependencies:
   ```
   npm install
   ```

3. Run the challenges:
   ```
   node src/index.js
   ```
